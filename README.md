# CircadianBro

App Android in Kotlin per impostare una sveglia e ricevere suggerimenti sugli orari migliori per andare a dormire in base ai cicli del sonno.

## Funzioni principali

- Scelta dell'orario di sveglia
- Suggerimenti per andare a dormire basati su cicli da 90 minuti
- Sveglia con suono e vibrazione
- Opzione per spegnere la sveglia risolvendo 3 operazioni matematiche
- Interfaccia minimal in tonalita viola e bianco

## Tecnologie usate

- Kotlin
- Android Studio
- Jetpack Compose
- AlarmManager
- SharedPreferences
